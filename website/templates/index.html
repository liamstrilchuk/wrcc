Language-specific

  • Java: You must have a class named "Main" in your file; otherwise, you will get a compilation error. A valid Java program looks like: 
    public class Main {
	public static void main(String[] args) {
		// your code here
	}
}
  • JavaScript: Since we use Node.js to mark JavaScript submissions, the prompt() function does not work. Instead, use the following format (or something similar): 
    const readline = require("readline");
const rl = readline.createInterface({
	input: process.stdin,
	output: process.stdout
});

let lines = [];

const start = async () => {
	for await (const line of rl) {
		lines.push(line);
	}
	// your code here
}
start();
